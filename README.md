## Usage

- Compile

```
javac -cp ./lib/gson-2.10.1.jar ./src/*.java
```

- Run the server

```
java -cp ./lib/gson-2.10.1.jar:. src.Server
```

- Open another (or multiple) shell(s) and run the client(s)

```
java src.Client
```
